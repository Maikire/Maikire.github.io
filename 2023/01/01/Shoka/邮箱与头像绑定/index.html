# hash of email

邮箱的 hash 值

通过加密算法获取

  • 使用 加密工具
  • 选择 哈希 / 散列，然后输入你的邮箱
  • 算法选择 MD5
  • 生成的结果就是我们想要的 hash 值

通过头像 url 获取

  • 可以看评论者的头像 url，比如 https://gravatar.loli.net/avatar/614d5030c7301b12be113b3f08114171?size=80&d=mp 其中 614d5030c7301b12be113b3f08114171 就是邮箱的 hash 值

自定义头像

  • 需要在 Gravatar 中添加自己的图片，使你的邮箱与你的头像绑定
  • 如果你修改了头像后发现没有更新，请不要慌张，因为 Gravatar 有几天的缓存期（大概七天），安静的等待吧～
