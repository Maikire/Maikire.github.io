# 框架设计

  • 角色状态类：角色的基础信息和基础行为（抽象类）
  • 角色的动画：因为动画参数的名字可能会改变，所以单独做一个类储存名字
  • 角色的移动方式：提供移动的方法，由输入控制类调用
  • 输入控制：处理玩家的输入方式，例如：摇杆、键鼠、手柄......（本章以这个摇杆为例）
  • 角色选择器（可选）：选中目标后显示选中特效
