# JSON

  • JSON жҢҮзҡ„жҳҜ JavaScript еҜ№иұЎиЎЁзӨәжі•пјҲJavaScript Object Notationпјү
  • JSON жҳҜиҪ»йҮҸзә§зҡ„ж–Үжң¬ж•°жҚ®дәӨжҚўж јејҸпјҢе®ғжҜ” XML жӣҙе°ҸгҖҒжӣҙеҝ«гҖҒжӣҙжҳ“и§Јжһҗ
  • JSON зӢ¬з«ӢдәҺиҜӯиЁҖпјҡJSON дҪҝз”Ё Javascript иҜӯжі•жқҘжҸҸиҝ°ж•°жҚ®еҜ№иұЎпјҢдҪҶжҳҜ JSON д»Қз„¶зӢ¬з«ӢдәҺиҜӯиЁҖе’Ңе№іеҸ°гҖӮJSON и§ЈжһҗеҷЁе’Ң JSON еә“ж”ҜжҢҒи®ёеӨҡдёҚеҗҢзҡ„зј–зЁӢиҜӯиЁҖгҖӮзӣ®еүҚйқһеёёеӨҡзҡ„еҠЁжҖҒпјҲPHPпјҢJSPпјҢ.NETпјүзј–зЁӢиҜӯиЁҖйғҪж”ҜжҢҒ JSON
  • JSON д»Ҙзү№е®ҡзҡ„ж јејҸи®°еҪ•жҹҗдәӣеҸҳйҮҸзҡ„еҖјпјҢеҸҜд»Ҙз”ЁжқҘз»ҷеҸҳйҮҸиөӢеҖјжҲ–и®°еҪ•еҸҳйҮҸеҖј
