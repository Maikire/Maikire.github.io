# Sqlite 助手

  • 这篇文章介绍了如何在 Unity 中使用 Sqlite
  • Sqlite 助手是对 Sqlite 数据库语言的封装，使用者可以绕过数据库语言，用更简单的方法操作数据库
  • 路径问题：
    • 数据库文件（.db）应该放到 StreamingAssets 文件夹中
    • 向 Sqlite 助手提供 StreamingAssets 中的路径，Sqlite 助手利用路径助手处理路径
