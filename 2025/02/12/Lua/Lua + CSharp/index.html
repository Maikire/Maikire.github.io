# C# 调用 Lua

  • 在安装 Lua 的文件夹中找到 lua54.dll 并导入 CSharp 项目
  • 使用 DllImport 导入 Lua 函数
  • 注意：宏定义无法导出到 dll 中，应该使用宏对应的函数（在 Lua 源码中查找即可）
